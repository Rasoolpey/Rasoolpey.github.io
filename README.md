# Rasoolpey.github.io

Personal academic portfolio website for Rasool Peykarporsan.

## Overview

This repository contains a static website built with:

- `index.html`
- `index.css`
- `index.js`

The site includes sections for work/projects, publications, skills, and personal background.

## Project Structure

- `images/Projects/`: Images used in the Work/Projects section
- `images/Papers/`: Figures used in the Publications section
- `images/Hobbies/`: Icons used in the Beyond Work section
- `fonts/`: Web fonts used by the site

## Run Locally

From the repository root:

```bash
python3 -m http.server 8080
```

Then open:

`http://localhost:8080`

## Deployment

This repository is intended to be served via GitHub Pages from the `main` branch.

## Notes

- Keep project-related assets under `images/Projects/`.
- Keep publication figures under `images/Papers/`.
- Keep hobby icons under `images/Hobbies/`.
